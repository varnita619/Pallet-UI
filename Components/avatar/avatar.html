Avatar

An avatar component can be used to display circular user profile pictures. Avatar can be used to portray people or objects. It supports images, icons, or letters.

Example of Round Avatars

Here are some avatar components for you with examples. If you want to use them in order to develop your apps with a faster pace just add these classes to your code. For different sizes you can use these classes:- avatar-xl avatar-lg avatar-md avatar-sm
